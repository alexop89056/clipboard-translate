# Clipboard Text Translate script in Python
#
### Description
This script was created to translate text from the clipboard by a specific button.
- Works on Windows
#
### Installation
In the file translate.py there are 2 variables (translate_button and language) whose values are standard and can be changed both in the source code and in the script itself
- git clone https://github.com/alexop89056/clipboard-translate
- cd clipboard-translate
- python -m pip install -r requirements.txt
- python tranlate.py
